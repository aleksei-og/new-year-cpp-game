# new-year-cpp-game

## О проекте
**TargetMotion** - это 2D-игра, написанная на C++ **(еще в 2021 году, дата с подтверждением есть на видео в правом нижнем углу)** с использованием фреймворка Qt. Игрок управляет ёлочкой, которая перемещается по арене и стреляет по появляющимся подаркам. Игра включает систему рекордов, разные уровни сложности и визуальные эффекты.

## Демонстрация игры
Смотрите видео gameplay.mp4

## Управление
- **W/A/S/D** - перемещение ёлочки по полю
- **Мышь** - прицеливание (курсор показывает направление выстрела)
- **ЛКМ** - стрельба по подаркам
- **Чем точнее прицел** - тем эффективнее попадание!

## Особенности игры

### Игровой процесс
- Управляйте ёлочкой, собирайте подарки
- Каждый подарок имеет шкалу здоровья
- За уничтоженные подарки начисляются очки
- Ограниченное время на уровень

### Системные особенности
- **2 уровня сложности**:
  - Низкая: 4 минуты на игру
  - Высокая: 3 минуты на игру
- **Система рекордов** с сохранением в файл
- **Кастомизированный курсор** для точного прицеливания
- **Анимация взрывов** при попадании
- **Физика столкновений** со стенами и препятствиями
- **Система здоровья** у целей (подарков)

### Технические особенности
- Полностью объектно-ориентированный код на C++
- Использование Qt Graphics Framework для рендеринга
- Система сигналов и слотов для коммуникации между объектами
- Кастомизированные QGraphicsItem для игровых объектов
- Таймеры для анимации и игровых событий

## Структура проекта

### Основные классы
- **Triangle** - класс ёлочки (игрового персонажа)
- **Bullet** - класс пули с расчётом траектории
- **Target** - класс подарков (целей)
- **Sprite** - класс анимации взрыва
- **CustomScene** - кастомизированная графическая сцена
- **Menu** - меню выбора сложности
- **Widget** - главное игровое окно

### Ресурсы
- Графические спрайты для ёлочки и взрывов
- Кастомизированный курсор-прицел
- Иконки для интерфейса

## Требования для сборки
- Qt
- Qt Creator (рекомендуется)
- Компилятор C++

## Сборка и запуск
1. Клонируйте репозиторий
2. Откройте файл `Igra.pro` в Qt Creator
3. Настройте пути к ресурсам при необходимости
4. Соберите проект (Build)
5. Запустите приложение
